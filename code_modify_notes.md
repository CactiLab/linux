# Code Modification Notes

## Format
```C
// GL [Notes] Notation
...
[// GL [Notes] Notation]
...
//-----
```

## Notation Meaning
### +
New code added.
### -
Original code is commented out.
### original
Below part is the original code.
### modify
Below part is the modified or added code, used with original.

## Notes Meaning
### For Debug Purpose (can be deleted)
- DEBUG
### For Code Modification
#### General
- CODE
#### Cred Protection
- CODE\_CRED
#### PAGA Key Management
- CODE\_PAGA
